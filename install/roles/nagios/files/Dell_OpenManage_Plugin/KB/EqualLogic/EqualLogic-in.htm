Warning

1

The status of the EqualLogic has changed to (notApproved).

Warning

Disk drive is not approved.

Replace with approved disk drive(s).

1

The status of the EqualLogic has changed to (failed).

Critical

Disk drive failure.

Replace failed disk drive(s). If none of the resolutions work, then contact Dell Support.

1

The status of the EqualLogic has changed to (too-small).

Critical

Disk drive is smaller than other drives in the member. The drive cannot be used in the member.

Replace the drive(s) with drive(s) that has the same size or a greater size than the installed drives. If none of the resolutions work, then contact Dell Support.

1

The status of the EqualLogic has changed to (unsupported-version).

Critical

Disk drive cannot use the firmware running on the member.

Install the proper version of firmware. If none of the resolutions work, then contact Dell Support.

1

The status of the EqualLogic has changed to (preempt-failed).

Critical

Disk drive has excessive errors.

Replace with healthy disk drive(s). If none of the resolutions work, then contact Dell Support.
